AS BIBLIOTECAS BASICAMENTE ESTÃO DIVIDIDAS EM :
BIBLIOTECAS DA FILA:
	fila.h
	element.h
	functions_show.h
	
	A fila basicamente é composta pelos elementos. Cada Elemento possui um ponteiro aonde aponta para um dado. Possui 		também dois ponteiros, um para o próximo ponteiro e o segundo para o ponteiro anterior.
	A functions_Show são funcões genericas de dump de endereços e dados

BIBLIOTECAS DO PROJETO SO:
	SO.h
	processo.h

	Os processos nada mais são que os dados para aonde os elementos da fila de procesos, que nada mais é que uma 		estrutura fila,irão apontar.

	O sistema operacional possui sistema de ciclos, ou seja, um processo pode demandar mais de um ciclo para ser 			concluido.
